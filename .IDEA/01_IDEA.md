## 주요 기능

### 프로젝트 관리

* 프로젝트 생성 / 조회 / 수정 / 삭제
* 프로젝트 참여자 등록 및 관리

---

### 테스트 케이스 관리

* 제목
* 내용
* 테스트 유형 (기능 / 성능 / 버그 / 기타)
* 담당자
* 마감일
* 테스트 케이스 상태

---

### 테스트 결과 관리

* 실행 날짜
* 테스트 결과 (성공 / 실패 / 중단)
* 실패 사유
* 증빙 파일 첨부 (이미지, 텍스트 로그 등)

---

### 테스트 케이스 상태 관리

* 기본 상태 제공

  * TODO: 아직 시작 안 함
  * IN_PROGRESS: 진행 중
  * VERIFYING: 검증 중
  * DONE: 완료
  * FAIL: 실패
* **프로젝트별 상태 커스터마이징 가능**

  * 상태 추가 / 수정 / 비활성화
  * 상태 순서 및 흐름 관리
* 상태 변경 이력 관리

  * 누가 언제 어떤 상태로 변경했는지 추적 가능

---

### 사용자 관리

* 사용자 생성 / 조회 / 수정 / 삭제
* 권한 관리 (관리자 / 일반 사용자 / 조회 전용)

---

### 회의에서 사용할 수 있는 화면

**회의용 대시보드**

* 프로젝트 전체 테스트 현황
* 테스트 케이스 상태별 현황
* 담당자별 진행 상황
* 마감일이 지난 테스트 목록
